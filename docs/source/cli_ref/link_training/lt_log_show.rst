lt log show
============

Description
-----------

To show the link training trace log for the specified lane.



Synopsis
--------

.. code-block:: console
    
    lt log show <LANE>


Arguments
---------

``<LANE>`` (integer)

Specifies the lane index.



Options
-------



Examples
--------

.. code-block:: console

    xoa_util[port0/2]$ lt log show 0
