anlt status
===========

Description
-----------

To show the current status of the working port.


Synopsis
--------

.. code-block:: console
    
    anlt status


Arguments
---------


Options
-------


Examples
--------

.. code-block:: console

    xoa_util[port0/0]$ anlt status
    
    Port 0/0
    Auto-negotiation        : on
    Link training           : interactive
    Link training timeout   : default
    Link recovery           : on
    Lane (serdes) count     : 2
