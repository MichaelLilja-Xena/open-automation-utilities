an log
======

Description
-----------

To start writing the auto-negotiation trace log into a file and display the log on the screen.



Synopsis
--------

.. code-block:: console
    
    an log <FILENAME>


Arguments
---------


``<FILENAME>`` (text)

Filename of the log.


Options
-------



Examples
--------

.. code-block:: console

    xoa_util[port0/2]$ an log "log0.txt"
    ...
    ...
    ...
