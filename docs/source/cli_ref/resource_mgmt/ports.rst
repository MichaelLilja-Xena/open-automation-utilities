ports
===================

Description
-----------

To list all the ports reserved by the current session.
This command works in all context.

Synopsis
--------

.. code-block:: console
    
    ports


Arguments
---------


Options
-------


Examples
--------

.. code-block:: console

    xoa_util[port0/0]$ ports
    Ports       Sync
    *0/0        yes
    0/1         yes
    0/2         yes

    xoa_util[port0/0]$
